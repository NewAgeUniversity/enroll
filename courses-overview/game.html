Curriculum

The curriculum for this course includes the following modules:

  • Week 1: Introduction to Game Development - Overview of the game development process, types of games, and essential skills required.
  • Week 2: Game Design Fundamentals - Understanding game mechanics, storytelling, and creating engaging gameplay experiences.
  • Week 3: Prototyping and Iteration - Techniques for rapidly prototyping game ideas and the importance of playtesting and iteration in game design.
  • Week 4: Introduction to Game Engines - Overview of popular game engines (Unity, Unreal Engine) and their basic features and functionalities.
  • Week 5: 2D Game Development - Techniques for creating 2D games, including sprite creation, animations, and level design.
  • Week 6: 3D Game Development - Introduction to 3D modeling, texturing, and implementing physics in 3D games using game engines.
  • Week 7: Game Programming Basics - Fundamental programming concepts using C# (for Unity) or C++ (for Unreal Engine) relevant to game development.
  • Week 8: Sound Design and Music - Importance of audio in games, techniques for creating sound effects, and integrating music into the gameplay experience.
  • Week 9: Game Monetization Strategies - Understanding different monetization models (free-to-play, premium, in-app purchases) and marketing strategies for games.
  • Week 10: Capstone Project - Develop a complete game prototype, integrating all elements learned throughout the course, and present it to the class.

Gain essential game development skills from design to programming, culminating in a hands-on capstone project that showcases your ability to create an engaging game.
