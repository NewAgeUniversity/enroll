NAU - Web Development

Web Development

Course Overview

The Web Development course at New Age University covers everything you need to build dynamic, responsive websites from scratch. You will learn front-end development with HTML, CSS, JavaScript, and back-end development using popular frameworks and databases. This course is designed for beginners and intermediate learners who want to pursue a career in web development or enhance their coding skills for personal projects.

Key Details

  • Duration: 6 months
  • Level: Beginner to Advance
  • Format: Online, Self-paced
  • Start Date: January 13th, 2025
  • Cost: Free

Curriculum

The curriculum for this course includes the following modules:

  • Week 1: Introduction to Web Development - Understanding how websites work and the tools you will need.
  • Week 2: Introduction to CSS and Respnsive Web Design - CSS fundamentals and techniques for responsive layouts.
  • Week 3: Advance HTML and CSS - Semantic HTML and advanced CSS layout techniques.
  • Week 4: Introduction to JavaScript - JavaScript fundamentals, including syntax and data types.
  • Week 5: Advance JavaScript and Front-End Libraries- ES6 features, asynchronous programming, and using libraries.
  • Week 6: Introduction to Back-End Development - Server-side basics with Node.js and Express.js.
  • Week 7: Advance Back-End Development - User authentication, API development, and security practices.
  • Week 8: Full-Stack Development - Integrating front-end and back-end technologies in applications.
  • Week 9: Advance Topics in Web Development - Exploring PWAs, GraphQL, web security and accesibility.
  • Week 10: Capstone Project - A Comprehensive project to showcase web development skills.

    • Gain essential web development skills from front-end to back-end, ending with a hands-on capstone project.
