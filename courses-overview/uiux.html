User Interface and User Experience

Course Overview

The User Interface (UI) and User Experience (UX) course at New Age University is designed to teach you the principles and practices behind creating intuitive, user-friendly digital products. From understanding user needs to designing seamless interfaces, this course equips you with the tools to build visually compelling and highly functional websites and apps. Ideal for aspiring designers, developers, and anyone interested in product design.

Key Details

  • Duration: 6 months
  • Level: Beginner to Advance
  • Format: Online, Self-paced
  • Start Date: January 13th, 2025
  • Cost: Free

Curriculum

The curriculum for this course includes the following modules:

  • Week 1: Introduction to UI/UX - Understanding the fundamentals of User Interface and User Experience design.
  • Week 2: Research and User Personas - Conducting user research and creating personas to guide design decisions.
  • Week 3: Wireframing and Prototyping - Building wireframes and interactive prototypes for design testing.
  • Week 4: Visual Design Principles - Exploring typography, color theory, and layout techniques for intuitive design.
