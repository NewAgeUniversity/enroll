5. Responsive Web Design

Responsive Web Design (RWD) is an approach to web design aimed at crafting websites that provide an optimal viewing experience across a wide range of devices. This involves using fluid grids, flexible images, and media queries to adapt the layout and content of a web page based on the device's screen size.

5.1. What is Responsive Web Design?

Responsive web design ensures that web pages render well on various devices and screen sizes. The goal is to provide a smooth and enjoyable user experience, regardless of whether the visitor is using a smartphone, tablet, or desktop computer.

5.2. Media Queries

Media queries are used to apply different styles depending on the device's characteristics, such as its screen width. This allows for a tailored design that adapts to various screen sizes.

@media (max-width: 768px) {
    body {
        background-color: lightblue;
    }
    .container {
        width: 100%;
    }
}

This media query sets the background color to light blue and adjusts the container width to 100% when the screen width is 768px or less. You can experiment with different breakpoints to target various device sizes.

5.3. Fluid Grids

Fluid grids use percentage-based widths rather than fixed pixel values. This allows layout elements to resize proportionally based on the screen size.

.column {
    width: 50%;
}

This column takes up 50% of the available width, adjusting dynamically as the screen size changes. You can use fluid grids to create flexible layouts that adapt to different screen sizes.

Flexible Images

Flexible images scale with the size of their container. This ensures that images do not overflow their containers and maintain their aspect ratio across different devices.

img {
    max-width: 100%;
    height: auto;
