Learn how to control text appearance using CSS properties such as font family, size, weight, and alignment.

2. Styling Text and Fonts

CSS provides numerous properties that allow you to manipulate and style text on web pages, including font choices, size, alignment, and spacing. Understanding these properties is essential for creating visually appealing websites.

2.1. Text Properties

CSS allows for extensive control over the appearance of text. The key properties to manipulate text include:

• Font Family

The font-family property specifies the font used for text. You can define multiple fonts in a list as a fallback in case the first font isn't available on the user's device.

body {
    font-family: 'Arial', sans-serif;
}

Explanation: The above example sets the font for the entire body to Arial, with sans-serif as the fallback font if Arial is unavailable.

• Font Size

The font-size property determines the size of the text. You can use various units such as pixels (px), ems, or rems to set the size.

p {
    font-size: 16px;
}

Explanation: This code sets the font size for all paragraph elements (<p>) to 16 pixels, which is a common default size for body text.

• Font Weight

The font-weight property controls the thickness or boldness of the text. Values can range from normal to bold, or you can specify numeric values like 400 (normal) or 700 (bold).

h1 {
    font-weight: bold;
}

Explanation: The example above sets the font weight of <h1> elements to bold, making the text thicker and more prominent.

• Text Alignment

The text-align property is used to align text horizontally within its container. Common values include left, right, center, and justify.

.centered-text {
    text-align: center;
}

Explanation: This CSS class centers the text within the container, often used for headings or important text elements.

• Line Height

The line-height property sets the vertical spacing between lines of text. A larger value increases the space between lines, improving readability.

p {
    line-height: 1.6;
}

Explanation: The example above sets the line height for all paragraph elements to 1.6 times the font size, adding extra spacing between lines of text.

2.2. Using Web Fonts

Web fonts are a great way to enhance the design of a website. Instead of relying solely on system fonts, you can use custom fonts by importing them from online sources like Google Fonts or Adobe Fonts.

Example: Importing Google Fonts

Here's how to import and use the "Roboto" font from Google Fonts:

@import url('https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap');

body {
    font-family: 'Roboto', sans-serif;
}

Explanation: This code imports the "Roboto" font from Google Fonts and applies it to the entire page by setting it as the font-family for the body element. The import URL specifies the font weights (400 for normal and 700 for bold) and ensures the font displays properly on all devices.
