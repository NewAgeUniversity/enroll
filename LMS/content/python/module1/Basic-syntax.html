Basic Syntax and Data Types in Python

Variables and Data Types

In Python, variables are containers for storing data values. You do not need to declare a variable explicitly; you can directly assign a value to a variable.

Data Types:

  • Integers: Whole numbers, e.g., 5, 100
  • Floats: Decimal numbers, e.g., 3.14, 2.0
  • Strings: Sequence of characters enclosed in quotes, e.g., "Hello, World!"
  • Booleans: Represents True or False

Python Code Examples:

coordinates = (10.0, 20.0)

In this example, coordinates is a tuple with two float values.

person = {"name": "Alice", "age": 25}

This is a dictionary with keys "name" and "age" and their corresponding values.

Type Conversion

Python provides built-in functions to convert between different data types.

num_str = "123"
num_int = int(num_str)  # Convert string to integer

In this example, the string "123" is converted to an integer 123.

Classic Introductory Code

As a tradition in programming, we often start with a classic "Hello, World!" program to familiarize ourselves with the basic syntax of the language.

print("Hello, World!")

This simple code prints "Hello, World!" to the screen and is often the first program written when learning a new programming language.
